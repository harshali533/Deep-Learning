# GenAI Assistant

A comprehensive Generative AI assistant featuring RAG, Agentic workflows, and Fine-tuning demos.

## Structure
- `app.py`: Streamlit UI
- `modules/`: Core logic for RAG, Agents, and LLM interaction
- `tools/`: Custom tools for agents
- `utils/`: Helper functions and logging

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Configure `.env` with your API keys.
3. Run the app: `streamlit run app.py`
